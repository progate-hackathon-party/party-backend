# タイムセールの場所と様子を投稿できる地図アプリ
> 必要な環境(*は推奨環境)
- ReactNative
- Makefile*
- Xcode/Simulator
- CocoaPods
- Node,npm
- brew
- watchman
<br>

> React Nativeのインストール
```
make install
```

> iOSビルド
```
make build_ios
```

> iOS Appを起動
```
make ios
```

<br>
ウィジェットや画面などは`./Widgets`フォルダに配置してください。<br>
開発には`Amplify`を使用します。<br>
